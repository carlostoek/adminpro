   - Add to session (no commit)
   - Return (True, "Story started", progress)

Logging:
- INFO: "Starting story {story_id} for user {user_id}"
- INFO: "Resuming story {story_id} for user {user_id}"
- ERROR: "Story {story_id} not found or not published"

<task type="auto">
  <name>Task 5: Implement make_choice method with immediate persistence</name>
  <files>bot/services/narrative.py</files>
  <action>
Add make_choice method to NarrativeService:

```python
async def make_choice(
    self,
    user_id: int,
    progress: UserStoryProgress,
    choice_id: int
) -> Tuple[bool, str, Optional[StoryNode], Optional[UserStoryProgress]]:
```

Purpose: Process a user's choice and advance the story. Persists progress immediately.

Logic:
1. Verify progress belongs to user_id
2. Fetch the choice with eager loading of target_node:
   - select(StoryChoice).where(StoryChoice.id == choice_id)
   - options(selectinload(StoryChoice.target_node))
3. Validate choice:
   - Must exist
   - Must be active
   - Must belong to current node (choice.source_node_id == progress.current_node_id)
4. If invalid: return (False, "Invalid choice", None, None)
5. Record decision:
   - Append to progress.decisions_made: {"choice_id": choice_id, "node_id": progress.current_node_id, "made_at": datetime.utcnow().isoformat()}
6. Advance to target node:
   - progress.current_node_id = choice.target_node_id
7. Check if reached ending:
   - If target_node.node_type == NodeType.ENDING:
     - progress.status = StoryProgressStatus.COMPLETED
     - progress.completed_at = datetime.utcnow()
     - progress.ending_reached = f"ending_{target_node.id}"
8. Update progress.updated_at
9. Add progress to session (dirty state, will be committed by handler)
10. Return (True, "Choice made", target_node, progress)

Logging:
- INFO: "User {user_id} made choice {choice_id} in story {story_id}"
- INFO: "User {user_id} completed story {story_id}, ending: {ending}"

Returns Tuple[bool, str, Optional[StoryNode], Optional[UserStoryProgress]]
  </action>
  <verify>grep -A 40 "async def make_choice" bot/services/narrative.py</verify>
