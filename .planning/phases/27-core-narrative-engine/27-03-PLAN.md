```python
async def validate_story(

    story_id: int
) -> Tuple[bool, List[str], Dict[str, Any]]:
```

Purpose: Validate story completeness before publishing.

Logic:
1. Fetch story with all nodes and choices (eager load)
2. Initialize errors list and info dict
3. Check 1: Story has at least one node
   - If nodes empty: errors.append("Story has no nodes")
4. Check 2: Story has exactly one START node
   - start_nodes = [n for n in nodes if n.node_type == NodeType.START]
   - If len(start_nodes) != 1: errors.append(f"Story must have exactly one START node (found {len(start_nodes)})")
5. Check 3: Story has at least one ENDING node
   - ending_nodes = [n for n in nodes if n.node_type == NodeType.ENDING]
   - If len(ending_nodes) == 0: errors.append("Story must have at least one ENDING node")
6. Check 4: All nodes (except ENDING) have at least one outgoing choice
   - For each node where node_type != ENDING:
     - If node has no active choices: errors.append(f"Node {node.id} has no outgoing choices")
7. Check 5: All choices point to existing nodes
   - For each choice: verify target_node_id exists in story nodes
8. Build info dict:
   - node_count: total nodes
   - choice_count: total choices
   - start_node_id: id of start node
   - ending_node_ids: list of ending node ids
   - unreachable_nodes: nodes not reachable from start (optional advanced check)

Return (len(errors) == 0, errors, info)

Logging:
- INFO: "Validated story {story_id}: {len(errors)} errors, {info['node_count']} nodes"

Returns Tuple[bool, List[str], Dict[str, Any]] - (is_valid, errors, info)

Logic:

   - If not found: return (False, "Story not found")
2. Check current status:
   - If status == StoryStatus.PUBLISHED: return (False, "Story is already published")
   - If status == StoryStatus.ARCHIVED and not force: return (False, "Cannot publish archived story without force")
3. Validate story:
   - Call self.validate_story(story_id)
   - If not valid and not force: return (False, f"Story validation failed: {errors}")
4. Update story status:
   - story.status = StoryStatus.PUBLISHED
   - story.updated_at = datetime.utcnow()
5. Return (True, "Story published successfully")

Logging:
- INFO: "Published story {story_id}"
- WARNING: "Force-published story {story_id} with validation errors"

Returns Tuple[bool, str]
