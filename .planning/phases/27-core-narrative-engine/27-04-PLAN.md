The ServiceContainer uses lazy loading pattern:
1. Private attribute initialized to None in __init__ (_service_name = None)
2. Property that checks if None, imports and instantiates if needed
3. get_loaded_services() method lists all loaded services
4. preload_critical_services() for eager loading critical services

Pattern example:
```python
def __init__(...):
    self._content_service = None

@property
def content(self):
    if self._content_service is None:
        from bot.services.content import ContentService
        self._content_service = ContentService(self._session)
    return self._content_service
```
